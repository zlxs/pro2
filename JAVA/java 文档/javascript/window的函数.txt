定时器
setTimeout()
  一次性定时器

clearTimeout()
  删除一次性定时器

-------------------------------
setInterval("表达式","时间") 或setInterval(function,"时间")
  循环定时器

例：
function test(){ 
    ......
}
var t1=setInterval(test,1000);//每秒执行一次test()函数


或直接var t1=setInterval(function(){ ... },1000);


如果需要页面在加载时立即执行，而不需要先等待1000毫秒，则可以先手动调用一次test函数


clearInterval(定时器对象)
  删除循环定时器

使用数据库链接池优化程序性能
数据库把链接存放在链接池中，用时从池中取，用完返回给链接池，而不是直接问数据库要链接

编写链接池需要实现DataSource接口


需要增强close()方法，使调用时，返回给连接池而不是给数据库

增强方法的三种方式：
1.写一个子类，覆盖原方法
2.用包装设计模式
3.用动态代理



用包装设计模式：
1.定义一个类，实现与被增强对象相同的接口，一般sun公司都有默认的接口用于包装设计模式
2.在类中定义一个变量，记住被增强对象
3.在类中定义一个构造函数，接收被增强对象
4.覆盖想增强的方法
5.对于不想增强的方法，直接调用目标对象（被增强对象）的方法

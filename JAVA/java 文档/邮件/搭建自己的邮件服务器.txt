需要在DNS上注册两条A记录和Mx记录（在网上买一个域名，域名商会给你一个域名管理程序，让你管理自己的域名，这时可以添加A记录和Mx记录）

注册A记录目的：
让其他邮件服务器转投邮件到你的服务器时，不需要这个服务器提供用户名密码（从aaa@qq.com到bbb@you.com）
普通用户发邮件需要用户名和密码(即发送方的账户)，而从一个服务器转发到另一个服务器则不需要，因为在DNS中有服务器的信息，收件方服务器会内部
得到发件方服务器的地址，再与DNS中的匹配，来确认发件方邮件服务器的信息是否属实，属实则不需要提供用户名和密码




注册Mx记录目的：
便于自己的邮件服务器能让管理发件方邮件的邮件服务器找到（从aaa@qq.com到bbb@you.com）
